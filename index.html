Enterprise IT Operations Agent: Autonomous Inventory Lookup & Automated CRM Ticketing

Built an end-to-end AI automation agent using Microsoft Copilot Studio to eliminate manual IT support bottlenecks. The agent orchestrates three critical enterprise workflows seamlessly in a unified chat interface: it serves as a semantic knowledge base for internal policies, performs real-time queries against SharePoint-hosted Excel inventory data to give instant device availability/pricing, and autonomously triggers custom APIs to generate CRM service desk tickets. By moving data lookup and ticket creation directly into the conversational canvas, this solution drastically reduces portal fatigue, manual data entry errors, and time-to-resolution for IT incidents.
